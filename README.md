# 🎓 Student Management System

## 📖 About
A simple **Student Management System** built using **Python** and **CSV File Handling**. This project helps manage student records through a user-friendly menu.

## ✨ Features
- ➕ Add Student
- 📋 View Students
- 🔍 Search Student
- ✏️ Update Student
- ❌ Delete Student
- 💾 Store Data in CSV File
- 🖥️ Menu-Driven Program

## 🛠️ Technologies
- 🐍 Python
- 📄 CSV Module
- 💻 VS Code

## 📂 Project Files
```
student_management.py
students.csv
README.md
```

## ▶️ How to Run
1. Open the project in **VS Code**.
2. Run **student_management.py**.
3. Choose an option from the menu.
4. Manage student records easily.
